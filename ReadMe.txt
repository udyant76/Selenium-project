Cucumber Example Tests for https://the-internet.herokuapp.com/
The implementation of testing of features of Heroku Internet Page used Java, Junit, Selenium and Cucumber.
Pre-requisites
Java 1.8.0 
Junit 4.12
Eclipse (IDE)
Selenium
Cucumber

External Jar files to be used for this purpose:
JUnit 4.12.jar
Cucucmber-java-1.2.2.jar
cucumber.junit-1.2.0.jar
cucumber-core-1.2.4.jar
selenium-java-2.47.0.jar
cucumber-jvm-deps-1.0.5.jar
cucumber-reporting-0.0.23.jar
mockito-all-1.10.8.jar
gherking-2.12.2.jar
cobertura-2.0.3.jar

Stes:

I recommend using Java SDK 1.8 for this purpose.
Install Java 1.8
Install Eclipse IDE
Dowload seleniumserver
download external jar files as mentioned above in folder.
Create a project in Eclipse and add external jar files in the buildpath.
Create a folder for java project and a folder for cucumber
Create scripts and run scripts using Junit.
